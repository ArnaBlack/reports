CSS Anchor Positioning API

Anchor

Элемент-ориентир. К нему привязываются другие

anchor-name: --my-anchor;

Target

Cпозиционированный элемент, который нужно разместить

position: absolute; /*or fixed*/
/*(Привязываем цель к якорю)*/
position-anchor: --my-anchor;

Якорь должен быть валидным

			 	target 

				<!-- ❌ Не работает! -->
				<div class="anchor">anchor</div>
			</div> 


			<div class="anchor">

				<!-- ✅ Так можно! -->
				<div class="target">target</div>
			</div>

Умные fallbacks:

position-try-*

			
				.target {
					position: fixed;
					position-try-fallbacks: --my-custom-position,
					--my-custom-position flip-block,
					bottom left;
				}
			
		
			
				.target {
					/*try-tactic*/
					position-try-fallbacks: flip-block;
				}
			
		
			
				.target {
					/*position-area*/
					position-try-fallbacks: top right, bottom left;
				}

Поведение

  • position: fixed — фолбэк срабатывает у края вьюпорта
  • position: absolute — фолбэк срабатывает у края предка
  • Для absolute предку нужно указать overflow, иначе фолбэки (position-try) не запустятся

Какой fallback выбрать?

			
				.target {
					position-area: top;
					position-try-fallbacks: flip-block,
											flip-inline, 
											flip-block flip-inline;
				}
position-try-order:
normal |
most-width |
most-height |
most-block-size |
most-inline-size

Кастомные fallbacks

@position-try

MDN 
          
          @position-try <dashed-ident> {
			  <declaration-list>
			}

declaration-list

  • position-area
  • position-anchor
  • align-self / justify-self / place-self
  • Inset property ( + anchor() )
  • Margin property
  • Sizing property ( + anchor-size() )
MDN 
          
			.target {
			  position: fixed;
			  position-anchor: --my-anchor;
			  position-area: top;
			  width: 150px;

			  position-try-fallbacks: --my-custom-position;
			}

			@position-try --my-custom-position {
  			  position-area: bottom;
			  width: 100px;
			}

Какой fallback сработал?

Проблема:
тултип умный, а стрелка — нет

Решение

@container anchored(fallback) 

			
				.target {
					/* тут стили для позиционирования */
					/* ... */
  					container-type: anchored;

					&::before {
						content: '▲';
						position: absolute;
						bottom: 100%;
					}
				}
			
		
			
				@container anchored(fallback: flip-block) {
					.target::before {
						content: '▼';
						bottom: auto;
						top: 100%;
					}
				}

Дополнительные возможности

Функция anchor-size()

значение для свойств width/ height/ inset/ margin; подстраивает размеры и отступы таргета под габариты якоря

Значение anchor-center

для свойств justify-self/ align-self/ place-self; центрирует таргет относительно якоря

Итоги и выводы Anchor Positioning API

  • Что это?
    Мощнейший API для позиционирования на чистом CSS
  • Что решает?
    Избавляет от тонн хрупкого JS-кода для поповеров, тултипов, меню
  • В чем преимущество?
    Производительность, надежность, декларативность
  • Когда использовать?
    Для пет-проектов и экспериментов — уже сейчас
    Для продакшена — пока рано (Статус спеки: W3C Working Draft)
